Standards

La page web ( http://www.open-std.org/JTC1/SC22/WG21/ ) du groupe de travail" JTC1/SC22/WG21 - The C++ Standards Committee - ISOCPP" permet d'accéder aux dernières informations concernant l'évolution du standard C++.
Les textes définitifs des standards officiels peuvent être achetés auprès de l'ANSI. Les versions préliminaires "draft" sont accessibles librement.
Attention, les normes ne sont pas des tutoriels, mais des documents très techniques. il faut connaitre le langage pour s'y retrouver !
  • Standard C++14 (draft)
    http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2014/n4296.pdf
  • Standard C++11 (draft)
    http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2012/n3376.pdf
  • C++98
    http://www.open-std.org/jtc1/sc22/open/n2356/
